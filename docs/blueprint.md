# **App Name**: SMPN 5 Attendance

## Core Features:

- Multi-role Authentication: Firebase Authentication for Admin, Headmaster, Teacher, Staff, and Student roles.
- QR Code Attendance: Attendance using a fixed QR/Barcode, validated with GPS and school radius.
- Time Validation: Server-side time validation for check-in/check-out.
- Real-time Notifications: Instant status notifications (success, out of radius/time, already checked-in).
- Admin Dashboard: Configuration for check-in/check-out times, location, QR generation, user management, and report exporting.
- Headmaster Approval: Interface for Headmasters to approve leave requests (Sick/Permit/Official Duty).
- User Attendance History: Personal attendance history for all users, with leave request submission and optional proof upload.

## Style Guidelines:

- Primary color: A deep indigo (#3F51B5) for a sense of trust and institutional reliability.
- Background color: Very light gray (#F0F2F5), providing a clean and modern feel, suitable for a light color scheme.
- Accent color: Teal (#009688) used sparingly for highlighting key actions and information.
- Body and headline font: 'Inter', a sans-serif typeface, chosen for its modern, neutral, and machined aesthetic, suitable for both headlines and body text.
- Mobile-first design with a floating bottom navigation bar (Home, Absen, Laporan, Pengaturan) on mobile and a responsive sidebar/top menu on web.
- Use clean and modern icons to represent each navigation item and feature.
- Subtle animations and transitions to enhance user experience and provide feedback.
